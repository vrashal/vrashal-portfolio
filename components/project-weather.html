01 — Overview

What We Built

A responsive, real-time weather dashboard that lets users search for any city worldwide and instantly view current conditions alongside a 7-day forecast. The UI dynamically changes its theme — background gradients, icons, and colours — based on weather conditions and time of day.

All data is fetched live from the OpenWeatherMap REST API, with smart caching to minimise API calls and a geolocation fallback to auto-detect the user's city on load.

04 — Process

Challenge & Solution

The Challenge

OpenWeatherMap's free tier has strict rate limits. Heavy user typing in the search box would exhaust the daily quota quickly, and stale cached data could mislead users.

The Solution

Implemented 300ms debounce on the search input and a time-stamped localStorage cache with a 10-minute TTL. Cache hits bypass the API entirely; stale entries trigger a fresh fetch silently in the background.
